Projects

SHRUB - The tech support chat bot

Live Demo Check this out on GitHub
SHRUB stands for Support Helper Utility Responsive Bot. She is intended to sit between consumers and human tech support for a computer company and handle frequently asked questions in an intelligent way. She uses IBM Watson Conversation and Discovery APIs, and thus is still currently learning how to do her job!
Stack used: React Node.js MongoDB Watson APIs

Travel Buddy - Budget your travel expenses

Live Demo Check this out on GitHub
Travel Buddy easily stores your travel budget information locally through your browser, or if you'd like to access your trip information on multiple devices, you can also save your progress to the server. This project was a proof-of-concept for me; I was testing saving information to the browser cache and then moving it to database storage upon user request.
Stack used: HTML CSS JQuery MongoDB Node.js

Dinner and a Beer - Perfect beer pairings for random recipes

Live Demo Check this out on GitHub
Dinner and a Beer allows the user to select which protein/entree type they'd like, and the app will suggest a random beer in a style that matches the recipe. Perfect for indecisive beer lovers looking to try new brews. Please note: this project is using free-tier APIs and thus can reach its request limits very quickly. If you're having trouble with the app, please try again later.
Stack used: HTML CSS JQuery Edamam API Brewery DB API
